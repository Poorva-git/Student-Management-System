# 🎓 Student Management System

A Python-based Student Management System built using **SQLite** for efficient management of student records, marks, attendance, report generation, and data export.

---

## 📌 Features

### 👨‍🎓 Student Management
- Add Student
- View Students
- Search Student
- Update Student
- Delete Student

### 📚 Marks Management
- Add Marks
- View Marks
- Automatic Total Calculation
- Percentage Calculation
- GPA Calculation

### 📅 Attendance Management
- Add Attendance
- View Attendance
- Automatic Attendance Percentage

### 🔐 Authentication
- Admin Login
- Student Login
- Logout

### 📄 Reports
- Student Report Card
- Topper Details
- Rank List

### 📊 Dashboard
- Total Students
- Highest GPA
- Average Attendance
- Students Passed
- Students Failed

### 📁 Export
- Export Student Data to CSV
- Export Student Data to Excel

---

# 🛠️ Technologies Used

- SQLite3

- CSV Module

# ▶️ How to Run

### Clone Repository

```bash
git clone https://github.com/Poorva-git/Student-Management-System.git
```

### Move into Folder

```bash
cd Student-Management-System
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run

```bash
python main.py
```
